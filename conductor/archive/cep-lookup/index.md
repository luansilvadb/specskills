# Track: Consulta CEP com ViaCEP

## Descrição
Implementar uma aplicação web para consulta de CEP brasileiro utilizando a API gratuita ViaCEP, com interface em HTML, CSS e JavaScript puro.

## Objetivos
- Criar interface intuitiva para consulta de CEP
- Integrar com API ViaCEP (https://viacep.com.br/)
- Exibir dados completos do endereço
- Tratar erros e CEPs inválidos
- Design responsivo e moderno

## Tecnologias
- HTML5 semântico
- CSS3 (Flexbox/Grid, animações)
- JavaScript ES6+ (fetch API, async/await)
- ViaCEP API (REST)
