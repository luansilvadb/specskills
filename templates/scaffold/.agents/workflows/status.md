# /status

Show the current status of all tracks and overall project progress.

## Usage

```
/status
```

## Output Includes

- Overall progress percentage
- Total tracks and tasks
- Active track details
- Next pending action
- Track-by-track breakdown

## Status Indicators

> [!NOTE]
> - [x] Completed
> - [~] In Progress
> - [ ] Pending
